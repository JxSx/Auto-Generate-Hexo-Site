#### 四、hexo配置

<font size=8><a href="https://hexo.io/zh-cn/docs/">Hexo官方文档</a></font>

##### 找个盘，新建文件夹（名字随便取），右键选择Git Bash，输入以下命令

###### 1.安装hexo插件套装
```bash
npm install hexo-cli -g
```

```bash
npm install hexo --save
```

###### 2.初始化hexo

```
hexo init  #初始化
npm install #安装依赖
```

生成的文件如下

```
.
├── _config.yml
├── package.json
├── scaffolds
├── source
|   ├── _drafts
|   └── _posts
└── themes
```

```
hexo g #生成静态页面
hexo s #部署到本地服务器http://localhost:4000/ 
```


[预览效果](http://localhost:4000/)
