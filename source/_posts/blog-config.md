# 为博客添加飞雪特效

[js源码](https://www.lanzous.com/i2iftxc)
提取密码:hudc

在foot.ejs加上这段代码

```javascript
<script src="/js/snowstorm.js" type="text/javascript"></script>
<script>
	snowStorm.snowColor = '#ffffff';       //雪花的颜色
	snowStorm.flakesMaxActive = 80;        //雪花的数量
	snowStorm.useTwinkleEffect = true;     //雪花在内部或外部闪烁
</script>
```
src填的是js文件的路径
